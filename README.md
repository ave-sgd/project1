# Рабочий айтрекер на 1 сайте (1 файл)

Сайт теперь полностью в одном файле: `index.html` (HTML + CSS + JS).

## Как запустить правильно

1. В папке проекта выполните:
   ```bash
   python3 -m http.server 8080
   ```
2. Откройте в браузере: `http://localhost:8080/index.html`
3. Нажмите **Запустить трекер** — сайт автоматически запросит доступ к камере.

## Почему кнопка могла "не работать"

При открытии файла двойным кликом (`file://`) некоторые браузеры ограничивают работу камеры и/или внешних скриптов.
Запуск через `http://localhost` решает это.
