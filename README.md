# Простой айтрекер для рекламного проекта

Минимальный веб-сайт для отслеживания взгляда пользователя через веб-камеру.

## Что умеет

- Запуск/остановка трекинга взгляда.
- Отрисовка точки текущего взгляда.
- Простая теплокарта поверх страницы.
- Базовые метрики: количество сэмплов и средние координаты.

## Как запустить

1. Откройте `index.html` в браузере (лучше Chrome).
2. Разрешите доступ к камере.
3. Нажмите **Запустить трекинг**.

## Технологии

- HTML/CSS/JavaScript
- [WebGazer.js](https://webgazer.cs.brown.edu/)

> В демо-версии данные не отправляются на сервер. При необходимости можно добавить экспорт или API.
